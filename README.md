# 🛒 PRODEX - React E-Commerce Application

A modern and responsive **E-Commerce Web Application** built with **React.js**. This project demonstrates core React concepts such as reusable components, Props, Prop Drilling, React Hooks, and REST API integration using **JSON Server**. It was developed to strengthen frontend development skills and simulate a real-world shopping experience.

---

## 🚀 Features

- 🛍️ Browse products
- 🔍 Product details page
- 🛒 Add to Cart
- ➕ Update cart quantity
- ❌ Remove items from cart
- 📱 Responsive user interface
- ⚡ Dynamic data rendering using API
- 🔄 State management with React Hooks

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Pavan-Raj-04/PRODEX-.git
```

### 2. Navigate to the project

```bash
cd PRODEX-
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the React application

```bash
npm start
```

### 5. Start JSON Server

```bash
npx json-server --watch src/jsondata/appdata.json --port 3001
```

> Update the port if your project uses a different one.
